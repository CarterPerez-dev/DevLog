# backend/api/tasks

## Purpose and Scope
The `backend/api/tasks` package in CertGames-Core is designed to handle asynchronous tasks using Celery. It encapsulates various task modules that perform operations such as mentorship management, referral processing, and data updates.

## File Relationships
- **celery.py**: Configures the Celery app and sets up task execution.
- **clean.py**, **mentorship.py**, **referral.py**, **update.py**: Contain specific tasks related to cleaning data, managing mentorship relationships, handling referrals, and updating records respectively.

## Public API Surface
The public API primarily includes:
- Task functions defined in each module file (e.g., `clean.clean_data`, `mentorship.assign_mentor`).
- Celery task objects configured in `celery.py`.

## When to Use This Module
Developers would interact with this package when they need to schedule or trigger background tasks. For instance, a developer might call `assign_mentor.delay(user_id)` from the `mentorship.py` module to asynchronously assign a mentor to a user.
