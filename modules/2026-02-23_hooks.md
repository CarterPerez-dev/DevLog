# Shared Hooks Module Documentation

## Purpose and Scope
The `frontends/react-native-ios/src/shared/hooks` module provides reusable React Native hooks for common functionalities in iOS applications, such as state management, biometric authentication, online status tracking, and session handling.

## File Relationships
- **index.ts**: Serves as the entry point, exporting all other hooks.
- **useAppState.ts**: Manages application state changes.
- **useBiometricAuth.ts** and **useBiometrics.ts**: Handle biometric authentication mechanisms like Touch ID or Face ID.
- **useOnlineManager.ts**: Tracks network connectivity status.
- **useSession.ts**: Handles user session management, including login and logout.

## Public API Surface
The public API includes:
- `useAppState`: Accesses application state changes.
- `useBiometricAuth`: Facilitates biometric authentication processes.
- `useBiometrics`: Provides access to biometric features.
- `useOnlineManager`: Monitors network status.
- `useSession`: Manages user sessions.

## When to Use
Developers should use this module when building React Native iOS applications that require state management, biometric security, online status tracking, or session handling. This ensures consistent and secure implementation across the application.
