## Purpose and Scope
The `frontend/src/core/lib/stores` package manages state management for various UI components in the `vuemantics` repository. It encapsulates store logic for authentication, bulk uploads, galleries, shell UI elements, and global batch progress.

- **auth.form.store.ts** and **auth.store.ts**: Handle user authentication states.
- **bulk-upload.ui.store.ts**, **gallery.ui.store.ts**, **shell.ui.store.ts**, and **upload.ui.store.ts**: Manage state for specific UI components related to bulk uploads, galleries, shell interfaces, and file uploads.
