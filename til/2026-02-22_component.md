## Today I Learned

TIL: In the `Component` function, the use of `<Link>` from React Router DOM for navigation is clever. It simplifies routing within the app compared to plain `<a>` tags, ensuring a smoother user experience and better state management.

```tsx

  Open Demo
</Link>
```

This pattern keeps your code clean and leverages React's powerful navigation capabilities.
