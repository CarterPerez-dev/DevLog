```python
async def __call__(self, message: Message, next_: SendFunc) -> None:
        self._next = next_
        self._buffer.append(message)

        if len(self._buffer) >= self._max_size:
            await self.flush()
            return

        if self._timer is None:
            loop = asyncio.get_running_loop()
            self._timer = loop.call_later(
                self._window,
                lambda: asyncio.ensure_future(self.flush()),
            )
```

## Today I Learned

TIL: `asyncio.get_running_loop()` and `loop.call_later` are used to schedule asynchronous tasks with precise timing within an async function. In `Batch.__call__`, this ensures that messages are flushed after a specified window, even if more messages are received before the timer triggers.

This pattern elegantly handles batching without blocking the event loop, making it both efficient and Pythonic.
